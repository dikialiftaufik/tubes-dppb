## ğŸš€ Cara Menggunakan

### **Step 1: Login / Register**
1. Buka aplikasi The Komars
2. Masuk dengan email dan password Anda
3. Jika belum punya akun, tap **"Register"** untuk membuat akun baru
4. Isi form registrasi dengan data lengkap
5. Klik **"Create Account"**

### **Step 2: Jelajahi Menu**
1. Dari halaman Home, tap **"Menu"** di bottom navigation
2. Menu akan ditampilkan dalam grid
3. Gunakan filter di atas untuk:
   - **Semua**: Lihat semua menu
   - **Sate**: Filter menu Sate saja
   - **Tongseng**: Filter menu Tongseng saja
4. Tap menu apapun untuk melihat detail lengkap

### **Step 3: Lihat Detail Menu**
1. Halaman detail akan menampilkan:
   - Deskripsi lengkap
   - Harga per porsi
   - Opsi untuk mengubah jumlah
2. Gunakan tombol **+ / -** untuk pilih jumlah porsi
3. Lihat total harga
4. Tap **"Tambah ke Keranjang"** untuk tambahkan ke cart

### **Step 4: Checkout**
1. Tap shopping cart icon di AppBar Menu
2. Review item keranjang Anda
3. Ubah jumlah atau hapus item jika perlu
4. Tap **"Lanjut ke Pembayaran"**
5. Pada halaman Checkout:
   - Review ringkasan pesanan
   - Pilih alamat pengiriman
   - Pilih metode pembayaran (Transfer Bank atau QRIS)
   - Lihat breakdown harga
6. Tap **"Konfirmasi Pesanan"**
7. Konfirmasi di dialog yang muncul
8. Pesanan Anda akan diproses âœ…

### **Step 5: Track Pesanan**
1. Dari halaman Profile, tap **"Pesanan Saya"**
2. Tab **"Aktif"**: Lihat pesanan yang sedang diproses
3. Tab **"Riwayat"**: Lihat pesanan yang sudah selesai
4. Tap **"Lihat Detail"** untuk melihat detail pesanan
5. Status pesanan akan diupdate secara real-time
